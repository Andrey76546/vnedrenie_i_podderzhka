### 1. Анализ функционала AIDA64
**Вывод (сравнение с аналогами):** 
AIDA64 Extreme Edition выгодно отличается от *PC Wizard* актуальностью баз данных и полноценной поддержкой современного оборудования. В отличие от *FurMark*, который является узконаправленным стресс-тестом исключительно для видеокарт, AIDA64 предоставляет комплексный анализ всей системы (ОЗУ, ЦП, диски, сеть). По сравнению с *PassMark Burning Test*, AIDA64 обладает более глубоким и наглядным инструментарием для мониторинга датчиков в реальном времени и сбора детальной информации об установленной операционной системе.

### 4. Информация о чипсете (Системная плата -> Чипсет)
В современных компьютерных системах архитектура материнских плат изменилась:
* **Северный мост:** Физически отсутствует как отдельная микросхема на материнской плате. Все его функции (контроллер оперативной памяти, контроллер шины PCI Express) интегрированы непосредственно в кристалл центрального процессора.
* **Южный мост:** Представлен микросхемой чипсета (PCH — Platform Controller Hub), которая отвечает за периферию (USB, SATA, звук, сеть).

## ❓ Ответы на контрольные вопросы

**1. Для чего необходимо выполнять стресс-тесты компьютерной системы?**
Стресс-тесты необходимы для выявления скрытых аппаратных неисправностей, проверки абсолютной стабильности системы при максимальных, пиковых нагрузках, а также для оценки эффективности установленной системы охлаждения процессора и видеокарты.

**2. Для чего используются программы диагностики компьютера?**
Они используются для получения исчерпывающей спецификации об аппаратном и программном обеспечении ПК, мониторинга состояния жизненно важных компонентов (температуры, напряжения, скорости вращения кулеров) и своевременного обнаружения возможных сбоев в работе системы.

**3. Какая из известных программ обладает наибольшим функционалом для тестирования ПК?**
Наибольшим функционалом для сбора информации и мониторинга датчиковдает программа **AIDA64**. Для комплексного тестирования производительности (бенчмаркинга) лидером является **PassMark PerformanceTest**, а для глубокого стресс-тестирования отдельных узлов (ЦП, питание)
