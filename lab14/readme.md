## ❓ Ответы на контрольные вопросы

### 1. Что такое драйвер?
Драйвер — это специализированная программа, которая позволяет операционной системе взаимодействовать с аппаратным обеспечением компьютера. Без корректно установленного драйвера устройство может работать некорректно, с ограниченным функционалом или не определяться системой вовсе.

### 2. Что такое ИД оборудования и его расшифровка?
ИД оборудования (Hardware ID) — это уникальная строка, присваиваемая устройству производителем, позволяющая точно идентифицировать его в системе.
* **VEN (Vendor):** Код компании-производителя устройства.
* **DEV (Device):** Код конкретной модели устройства.
Данные параметры необходимы для поиска и загрузки подходящего драйвера на официальном сайте производителя.
