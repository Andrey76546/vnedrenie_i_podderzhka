## ❓ Ответы на контрольные вопросы

1. **Что такое безопасный режим Windows?**
Это способ загрузки операционной системы, при котором запускаются только минимально необходимые компоненты, без загрузки приложений автозагрузки, что позволяет диагностировать вирусы, ошибки драйверов или нестабильное ПО.


2. **Параметры запуска Безопасного режима:**
Основные параметры включают «Минимальная», «Другая оболочка», «Восстановление Active Directory» и «Сеть».


3. **Отличия безопасного режима от чистой загрузки ОС:**
* **Безопасный режим** — это диагностический режим, который загружает только базовый набор драйверов и служб, необходимых для запуска ОС, и блокирует большинство сторонних приложений.
* **Чистая загрузка** — это режим, при котором ОС загружается в обычном режиме, но с отключенными сторонними службами и программами автозагрузки, что позволяет выявить конфликты ПО без перехода в упрощенный графический интерфейс безопасного режима.
