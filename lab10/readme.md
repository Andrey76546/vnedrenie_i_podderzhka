## 🎯 Задание 1. Техническое задание для ГК «BiCo»

### 1. Назначение и цели создания системы
Целью внедрения является повышение уровня безопасности офисных помещений ГК «BiCo» путем замены устаревшей системы контроля доступа на современную автоматизированную систему (СКУД).

### 2. Характеристика существующей системы (Старый парк ПК)
В качестве примера старой системы взята текущая рабочая станция IT-отдела:
* **Процессор:** Intel Core i5 (устаревшее поколение)
* **ОЗУ:** 8 GB DDR4
* **Накопитель:** 256 GB SATA SSD
* **Статус:** Требуется модернизация для поддержки нового ПО.
