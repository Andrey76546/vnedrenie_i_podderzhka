## 🛠 Ход выполнения работы
### 1. Вход в BIOS/UEFI
Для входа в систему BIOS были использованы соответствующие клавиши управления (Del, F2 или F10 при включении ПК). Изучен интерфейс и основные разделы: Main, Advanced, Boot, Security.

 * **Приоритет загрузки (Boot Priority):** Установлен порядок загрузки (USB-накопитель → SSD-накопитель). Данная настройка необходима для установки ОС или восстановления системы.
 * **Безопасность:** Установлен пароль администратора (Administrator Password) на вход в BIOS для предотвращения несанкционированного изменения системных параметров.
 * **Производительность:** Проверены настройки режима работы контроллера накопителей (выбрано AHCI для оптимальной работы SSD) и настройки энергопотребления процессора.

## 📊 Таблица основных разделов BIOS/UEFI
| Раздел | Назначение | Основные настройки |
|---|---|---|
| **Main** | Общая информация | Время, дата, версия BIOS, тип процессора |
| **Advanced** | Расширенные настройки | Настройка портов, контроллеров, периферии |
| **Security** | Безопасность | Установка паролей на вход, Secure Boot |
| **Boot** | Параметры загрузки | Приоритет устройств (HDD/SSD/USB) |
| **Exit** | Выход | Сохранение изменений или сброс к default |
## ❓ Ответы на контрольные вопросы
 1. **Что такое BIOS и зачем он нужен?**
   BIOS (Basic Input/Output System) — это базовое ПО, которое загружается при включении компьютера. Оно проверяет исправность оборудования (POST), инициализирует устройства и передает управление загрузчику операционной системы.
 2. **В чем разница между BIOS и UEFI?**
   UEFI (Unified Extensible Firmware Interface) — современная замена BIOS. UEFI поддерживает диски большого объема (более 2 ТБ), имеет графический интерфейс, мышь, поддерживает сетевую загрузку и обладает более высоким уровнем безопасности.
 3. **Что будет, если сбросить настройки BIOS?**
   Настройки вернутся к значениям «по умолчанию» (factory defaults). Это полезно при возникновении ошибок конфигурации, но потребует повторной настройки специфических параметров (например, приоритета загрузки или режима AHCI).
