## ❓ Ответы на контрольные вопросы

**1. Какие другие программы используют для тестирования видеокарт?**
Для комплексного бенчмаркинга и стресс-тестирования видеокарт также применяются:
* 3DMark (Time Spy, Fire Strike)
* Unigine Heaven / Unigine Superposition
* MSI Kombustor
* OCCT (раздел теста GPU)

**2. Какие программы используют для тестирования мониторов?**
Для проверки цветопередачи, контрастности, времени отклика и наличия дефектных (битых) пикселей используют:
* PassMark MonitorTest
* TFTTest
* AIDA64 (встроенный модуль Monitor Diagnostics)
* EIZO Monitor Test (веб-утилита)
